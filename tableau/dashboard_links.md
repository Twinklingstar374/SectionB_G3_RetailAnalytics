## 📌 Dashboard Overview

**Sales & Revenue Overview** | Provides insights into total revenue, order trends, payment methods, and  regional performance    
**Customer Insights Overview** | Analyzes customer demographics, segmentation, retention, and acquisition trends                  
**Product Performane Analysis** | Highlights top-performing products, brands, and categories based on revenue and ratings          
**Operational Insights Overview**| Focuses on order success rate, delivery performance, shipping methods, and feedback distribution 


##  Key Features

* Interactive filters for dynamic analysis
* Navigation across dashboards for seamless exploration
* KPIs for quick business understanding
* Visual insights using charts, treemaps, and maps
* Business-driven key insights included in each dashboard

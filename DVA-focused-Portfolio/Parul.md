# DVA-Focused Portfolio

This repository contains a portfolio-ready case study developed as part of the Data Visualization & Analytics (DVA) capstone project. The goal is to transform raw retail data into meaningful insights through interactive dashboards and structured analysis.

## Live Portfolio

👉 [View Portfolio](https://dva-resume-three.vercel.app/)

## Case Study Overview

1. Problem Statement

Retail businesses often struggle to gain a unified view of sales, customers, products, and operations. This project aims to provide a comprehensive analytics solution to support data-driven decision-making.

2. Dataset Source & Scope

* Retail dataset (academic capstone dataset)
* Includes: Orders, Customers, Products, Revenue, Shipping, Ratings
* Covers multi-dimensional analysis across sales, customer behavior, product performance, and operations

3. Data Cleaning & Transformation

* Removed null values (e.g., Month, Product Brand)
* Created calculated fields (Order Success Rate, AOV, Customer Segments)
* Standardized categorical fields
* Aggregated KPIs for dashboard reporting

4. KPI Framework

* Total Revenue
* Total Orders
* Average Order Value (AOV)
* Customer Retention Rate
* Order Success Rate
* Top Product Categories & Brands

5. Key Insights

* Revenue shows seasonal peaks (April, August)
* Regular customers contribute the highest share (~48%)
* Weekday sales exceed weekend performance
* Same-day shipping generates the highest revenue
* Customer retention rate is strong (~94%)

6. Tableau Dashboards

The analysis is presented through four interactive dashboards:

1. **Sales & Revenue Overview**
2. **Customer Insights Overview**
3. **Product Performance Analysis**
4. **Operational Insights Overview**

👉 [View Tableau Dashboard](https://public.tableau.com/app/profile/bulbul.agarwalla/viz/SectionB_RetailAnalysis_G3/ProductPerformanceAnalysis?publish=yes)

7. Recommendations & Impact

* Focus on retaining high-value and regular customers
* Optimize same-day delivery operations
* Improve processing and pending orders to reduce delays
* Leverage seasonal peaks for targeted marketing campaigns
* Expand strategies for high-value customer growth
